# **App Name**: Lumen NG

## Core Features:

- Regional Intake Portal: A streamlined mobile-responsive interface for registered medical facilities to report live birth data securely.
- State-Wise Analytics Dashboard: Visual representation of birthrate trends across Nigeria’s 36 states and the FCT using interactive choropleth maps.
- Census Database Management: A centralized, scalable storage system to maintain high-integrity birth records and facility information.
- Live Stats Ticker: A real-time updated ticker showing the estimated number of births across the nation for the current day and month.
- Continuous Birthrate Counter: A high-visibility digital odometer-style counter that continuously increments in real-time to represent national population growth.
- Automated Summary Reporting: Generates PDF-based regional census summaries monthly for health administrative review.
- Registration Officer Dashboard: A summary view showing today's registrations, pending certificate generations, and quick-access links to new registration and search functions.
- Birth Registration Form: A multi-section, paginated form organized into Child Details, Birth Event Details, Mother's Details, Father's Details, and Notifier Details sections with inline validation.
- Certificate Preview and Download: A confirmation screen displaying a preview of the generated birth certificate with a prominent download button.
- Parent Self-Service Portal: A simple interface where parents enter their child's registration number and their own details to retrieve and download the birth certificate.
- Administrator Dashboard: An analytics-focused view with bar charts for births by state, line graphs for monthly trends, and tabular drill-down reports with filter controls.
- User Management Screen: A tabular list of registered users with controls to create, edit, activate, or deactivate accounts.

## Style Guidelines:

- Primary color: A vibrant Emerald Green (#2DDC8F) chosen to evoke national identity and growth-oriented data clarity.
- Background color: A deep, desaturated Forest Green (#0F1A15) creating a sophisticated dark-mode dashboard feel.
- Accent color: A soft Spring Green (#98FF98) used for high-contrast indicators and focused call-to-actions.
- Headline and navigation font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and data-driven aesthetic.
- Body text font: 'Inter' (sans-serif) to ensure maximum legibility for dense statistical information.
- Sharp, geometric line icons using a 1.5px stroke weight to maintain a clinical and objective visual language.
- A card-based masonry grid layout that prioritizes high-level statistics first, followed by granular regional drills.
- Smooth, rolling-digit animations for the continuous birthrate counter to visualize real-time growth momentum.
